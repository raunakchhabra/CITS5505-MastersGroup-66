## Setup
1. Clone: `git clone https://github.com/raunakchhabra/CITS5505-MastersGroup-66.git`
2. Activate virtual environment: `source pyvenv/bin/activate`
3. Install: `pip install -r requirements.txt`
4. Run: `python3 run.py`
